# WeAnalyz Helpdesk App

A simple React-based Helpdesk UI built for internship assignment.

## Features

- Create ticket (form + local storage)
- View my tickets
- Approve tickets
- Dashboard with stats
- Clean Figma-based UI

## Tech Stack

- ReactJS
- CSS (custom)
- React Router
- LocalStorage
